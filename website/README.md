# QUARK ORM Website

This website is built using [Docusaurus](https://docusaurus.io/) and documents
the QUARK ORM under `pkg/quark`.

## Installation

```bash
npm install
```

## Local Development

```bash
npm start
```

This command starts a local development server. Most changes are reflected live
without having to restart the server.

## Build

```bash
npm run build
```

This command generates static content into the `build` directory and can be
served using any static hosting service.

## Deployment

Configure `url` and `baseUrl` in `docusaurus.config.ts` for the final hosting
target before publishing.
